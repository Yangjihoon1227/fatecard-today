# FateCard.today v27

## Added

- Selected cards can be clicked/tapped again before revealing to cancel the choice and put them back into the spread.
- Every tarot reading menu now has an "Ask AI tarot" question box.
- The question is combined with selected cards, reading type, card order, and profile data to generate a tailored answer block.
- The reading copy remains local/client-side and does not call an external AI API yet.
- Future server-side LLM integration can replace the local symbolic answer engine without changing the UX.

## Notes

The front card images continue to use the existing public-domain 1909 RWS route. The card back remains original site artwork.
