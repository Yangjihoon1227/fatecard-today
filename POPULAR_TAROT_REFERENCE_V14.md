# FateCard.today v14 popular tarot site reference

This version follows the feature patterns seen across major tarot destinations:

- Daily tarot / card of the day
- Free 3-card reading
- Love tarot
- Yes / No tarot
- Birth card
- 78-card meanings guide
- Tarot spreads
- Learning / quiz
- Journal / reflection
- Prompt builder
- Community and update sections
- Sponsor/contact funnel

The goal is not to copy a competitor's design. The goal is to use the proven information architecture while keeping FateCard's one-card-per-day hook.
