# FateCard.today v25

- First visit opens a polished home/landing screen.
- One sidebar only. Hamburger toggles collapse/expand instead of opening a second menu.
- Home icon and FateCard.today brand return to the home screen.
- Every reading uses a full-deck spread style: users choose card count and select cards from the spread.
- Card count selector supports 1, 3, 5, and 7 cards.
- Interpretation appears only after all selected cards are chosen.
- Card back uses original, non-commercial, tarot-inspired artwork.
- Front card images still use the public-domain 1909 RWS route already used in v23/v24.
