# FateCard.today v12 verification

## Copy
The main headline was shortened into stable two-line copy.

Examples:
- English: `One card.` / `Today’s fate.`
- Korean: `하루 한 장,` / `오늘의 운명카드.`
- Japanese: `一日一枚。` / `今日の運命カード。`
- Chinese: `每天一张。` / `今日命运卡。`

## Layout rule
The headline uses explicit line spans, so it does not randomly wrap because of one or two extra characters on desktop/tablet widths.
