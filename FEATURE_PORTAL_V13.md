# FateCard.today v13 Feature Portal

This version turns the site from a single daily-card page into a real tarot feature portal.

## Working sections

- Daily Fate Card
- Free 3-card tarot test
- 78-card searchable guide
- Tarot spreads
- AI-style interpreter
- Tarot prompt builder
- Community vote
- Update news
- Sponsor/contact

## Notes

The AI-style interpreter is deterministic and uses the 78-card deck data. It does not call an external AI API yet.
Community votes are stored in localStorage for now. Production should move this to a database.
Nano Banana-style visual direction is implemented with SVG/CSS assets and prompt packs.
