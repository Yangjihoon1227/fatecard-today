# FateCard.today v23 — public-domain tarot card artwork

This version uses the Wikimedia Commons `Rider-Waite tarot deck (Roses & Lilies)` image set through `Special:FilePath`.

## Why

The previous SVG symbolic cards were too abstract and did not feel like real tarot cards.

## Source strategy

- Use the original 1909 Rider–Waite–Smith / Roses & Lilies image set hosted on Wikimedia Commons.
- Avoid modern commercial recolors, boxed set images, proprietary card backs, and new edition designs.
- Keep the custom site UI, card back, layout, reading text, and product design separate from any modern commercial deck.

## Important

No legal setup can honestly be called 100% risk-free. This is a much more credible public-domain artwork approach than vague symbolic SVG cards.
